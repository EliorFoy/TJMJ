📝 Abstract

Taojiang Classic Wang Mahjong (TJMJ) is a full-stack web-based mahjong gaming platform designed for the Taojiang region of Hunan Province. The project fully implements the core rule system — dynamic wildcard (Ban Wang), Zha Niao multiplier scoring, Menzi cumulative fan patterns — supporting single-player AI, four-player online, and spectator modes. The architecture uses Vue 3 + Vite 8 frontend, Node.js WebSocket server, and Agora SD-RTN cloud voice (10,000 min/month free). Frontend on GitHub Pages, backend via ngrok TCP tunnel. Core algorithms include an O(3k) recursive backtracking winning-hand engine (sub-ms runtime), heuristic NPC strategy, and count-difference hand-tile sync. Fully open-source (~4,200 lines), with bilingual manuals: Chinese (30pp) and English (29pp).
