📝 Abstract

Taojiang Classic Wang Mahjong (TJMJ) is a full-stack web-based mahjong gaming platform designed for the Taojiang region of Hunan Province. The project fully implements the core rule system of Taojiang local mahjong — including dynamic wildcard determination (Ban Wang), Zha Niao multiplier scoring, Menzi cumulative fan patterns, and combination winning hands — supporting single-player AI, four-player online multiplayer, and spectator modes. The technical architecture adopts a Vue 3 Composition API frontend with Vite 8 build tooling, a Node.js WebSocket server, and Agora SD-RTN cloud-based real-time voice (10,000 min/month free tier). The frontend is deployed on GitHub Pages, with the backend exposed via ngrok TCP tunneling. Core algorithms include an $\mathcal{O}(3^k)$ recursive backtracking winning-hand detection engine (sub-millisecond measured runtime), a heuristic NPC discard strategy, and a count-difference client-server hand-tile synchronization algorithm. The project is fully open-source (MIT License) with approximately 4,200 lines of code, accompanied by complete technical manuals in both Chinese (30 pages) and English (29 pages).

✨ Key Features

🀄 Complete Taojiang Rules

Dynamic wildcard (Ban Wang) · Zha Niao multiplier scoring · Menzi cumulative fan patterns · Combination winning hands · 108-tile compact deck, faithfully matching Taojiang local play.

🌐 Full-Stack Web Architecture

Vue 3 Composition API + Vite 8 frontend · Node.js ws server · JSON over WebSocket real-time communication · GitHub Pages deployment.

🎙️ Real-Time Voice

Agora SD-RTN cloud relay · Opus 48kHz codec · 10,000 min/month free tier · No P2P NAT issues · Auto channel join on room entry.

🤖 Intelligent NPC

Heuristic AI discard strategy: ting-priority evaluation · hand-tile weighted scoring · defensive play · smart chi/peng decisions.

📱 Cross-Platform

iOS Safari adaptation · Android Chrome dvh support · Desktop scaling · Touch drag-and-drop · Auto-reconnection · 15s action timeout deadlock prevention.

🔧 Developer-Friendly

Fully open-source · Comprehensive bilingual documentation · TiKZ architecture diagrams · Algorithm pseudocode · Complete protocol tables · One-click launch script.
