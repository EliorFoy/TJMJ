📝 摘要

桃江经典王麻将 (TJMJ) 是一款面向湖南桃江地区的全栈网页麻将游戏平台。项目完整实现了桃江地方麻将的核心规则体系——包括打筛扳王（癞子动态确定）、扎鸟翻倍计分、门子累加番型、组合胡法等复杂机制，提供单机 AI、四人联机、观战三种游戏模式。 技术架构采用 Vue 3 Composition API 前端 + Vite 8 构建、Node.js WebSocket 服务端、WebRTC P2P 实时语音（辅以 Twilio TURN 中继），前端部署于 GitHub Pages，后端通过 ngrok 公网穿透。 核心算法包括 $\mathcal{O}(3^k)$ 递归回溯胡牌判定引擎（实测亚毫秒级）、启发式 NPC 出牌策略、以及计数差分客户端服务端手牌同步算法。 项目完全开源（MIT License），总代码量约 4200 行，已发布完整中英文技术手册（18 页）。

✨ 核心特性

🀄 完整桃江规则

打筛扳王 · 扎鸟翻倍 · 门子累加番 · 组合胡法 · 108 张精简牌组，与桃江本地玩法高度一致。

🌐 全栈 Web 架构

Vue 3 Composition API + Vite 8 前端 · Node.js ws 服务端 · JSON over WebSocket 实时通信 · GitHub Pages 部署。

🎙️ 实时语音

WebRTC P2P 全连接网状拓扑 · Opus 48kHz 编码 · Twilio TURN (10GB/月) NAT 穿透 · 增量 PeerConnection 管理。

🤖 智能 NPC

启发式 AI 出牌策略：听牌优先级评估 · 手牌价值加权排序 · 防守策略 · 吃碰智能决策。

📱 跨平台兼容

iOS Safari 适配 · Android Chrome dvh 适配 · 桌面端缩放 · 触屏拖拽 · 断线重连 · 10s 操作超时防死锁。
